Colour Schemes

Monochromatic

Monochromatic colour scheme are simply schemes with only 1 hue. As such, colour variety can only come from value and saturation. It is easy to use, but can feel limiting.

analogous

In this colour scheme, We have a little more options with our hues. the neighbouring colours in the colour wheel are used. It is similar to monochromatic but bring more variety.

complimentary

Now we start getting into spicier palattes. Here, the colour opposite our first colour is used. The important thing to know here is that one colour should ideally be more dominant, and have the other colour simply compliment it.

split complimentary

Similar to complimentary, but now we use the neighbouring colours.

triadic

Triadic is typically used for more playful and vibrant feelings. The colours should form an eqilateral triangle.

tetradic/double split complimentary

You can think of this as a split complimentary palatte but with both sides split. It's a little rare but definitely an option to be aware of.

square

Kinda self explanatory. It's like triadic but now a square instead of a triangle.
